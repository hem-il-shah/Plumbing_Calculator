# Sintex Plumbing Contractor Calculator

A two-part app (Express backend + Vite/React frontend) that builds an
estimated plumbing quotation from the Sintex master-data workbook
(`backend/data/Plumbing_Calculator.xlsx`) and lets you download a partitioned
PDF quotation.

## Flow

1. **Select System** — CPVC / CPVC + UPVC / Drainage (SWR)
2. **No. of Bathrooms** — integer, min 1
3. **No. of Kitchens** (CPVC / CPVC+UPVC) **or No. of Floors** (SWR)
4. **Bathroom Dimensions** — Length × Width in feet (base 8 × 4)
5. **Piping System** (CPVC / CPVC+UPVC only) — Single Lever Diverter/Flush Valve
   or Wall Mixer Tap
6. **Calculate** → Material-Group breakdown + downloadable PDF

## Pricing model

- Base reference room: **8 ft × 4 ft**
- Dimensional adjustment: **±1.5% per ft per axis** (so +2 ft on both
  axes = +6%)
- Final multiplier = dimension adjustment × bathrooms × (kitchens **or** floors)
- Items from the workbook are scaled by the final multiplier; PDF and on-screen
  partition by Material Group

## Running locally

Backend (port 4000, 127.0.0.1 only):

```
cd backend
npm install
npm start
```

Frontend (port 5173, 127.0.0.1 only — proxies /api to the backend):

```
cd frontend
npm install
npm run dev
```

Open http://127.0.0.1:5173/

## Quotation ID format

`ddmmyyyyhhmmssSSS<ip-digits><XXXX>` — single continuous token, e.g.
`11062026031908057192168142PZGC`. Generated by the backend so the IP
component reflects the client's actual address.
